1 - Les requêtes SQL

  • Créer une BDD : CREATE DATABASE IF NOT EXISTS nomBdd
  • Supprimer une BDD : DROP DATABASE IF NOT EXISTS nomBdd
  • Observer la description d'une table : DESC nomTable
  • Voir les BDD dans le serveur : code SHOW DATABASE

2 - Les tables et colonnes

  • Créer ne BDD 'entreprise ': CREATE DATABASE IF NOT EXIST entreprise
  • Créer une table ldans la BDD entreprise : 
    CREATE TABLE IF NOT EXISTS employes(
                              id_employes INT(11) NOT NULL PRIMARY KEY AUTO_INCREMENT,
                              prenom VARCHAR(20) NULL,
                              nom VARCHAR(20) NULL,
                              sexe ENUM ('f', 'm') NOT NULL,
                              service VARCHAR(30) NULL,
                              date_embauche DATE NULL,
                              salaire FLOAT NULL
                            )

3 - SELECT et DINSTINCT

  • Pour voir tous les contenu d'une table: SELECT * FROM employes (*=tous)
  • Sélectionner les valeurs de deux champs d'une table: SELECT colonne1 colonne 2 FROM nomTable SELECT prenom, nom FROM employes
  • Affichr les données sans doublons : SELECT DISTINCT (colonne) FROM nomTable SELECT DISTINCT (SERVICE) FROM employes
  • Supprimer un enregistrement: DELETE FROM nomTable WHERE DELETE FROM employes WHERE prenom 'Salimata'
